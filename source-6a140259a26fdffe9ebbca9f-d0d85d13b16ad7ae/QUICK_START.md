# Краткая инструкция - быстрый старт

## 🚀 За 3 минуты до PDF

### На Windows:
```bash
1. Дважды кликните на build.bat
2. Подождите ~2 минуты на первый раз
3. Файл готов: mila_guide_spring_2026.pdf
```

### На macOS/Linux:
```bash
chmod +x build.sh
./build.sh
```

### На любой системе через браузер:
```bash
1. Откройте mila_guide_pdf.html в браузере
2. Нажмите Ctrl+P (Cmd+P на Mac)
3. "Сохранить как PDF"
4. Параметры:
   - Поля: Нет (None)
   - ✅ Графический фон
   - ✅ Размеры страницы
5. Сохранить
```

## ⚠️ Если что-то не работает

**Браузер не сохраняет водяной знак при печати?**
- Используйте скрипт (более надёжно)

**Скрипт ошибается?**
- Установите Node.js: https://nodejs.org
- Переустановите: `npm install`

**Размер PDF слишком большой?**
- Сожмите на smallpdf.com
